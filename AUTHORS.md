## Acknowledgments

Solis draws inspiration from:

- **Haskell** - Lazy evaluation, monadic do-notation, functional programming patterns
- **ML family languages** - Hindley-Milner type inference
- **Lean/Agda** - Bidirectional type checking concepts
- **Other functional languages** - Pattern matching, algebraic data types

Special thanks to the functional programming community for the theoretical foundations and practical insights that inform Solis's design.
